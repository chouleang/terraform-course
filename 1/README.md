# Day 1: Terraform Fundamentals & Core Concepts

## Overview

This directory contains the Day 1 hands-on lab for learning Terraform basics using AWS.

**Duration:** 5-6 hours  
**Cloud Provider:** AWS  
**Region:** us-east-1

## What You'll Learn

- ✅ Infrastructure as Code (IaC) concepts
- ✅ Terraform installation and setup
- ✅ HCL syntax and configuration blocks
- ✅ Providers and authentication
- ✅ Resources and data sources
- ✅ The Terraform workflow (init → plan → apply → destroy)
- ✅ State file management

## Topics Covered

### Terraform Fundamentals
- What is Terraform?
- Why use Infrastructure as Code?
- Terraform vs other tools comparison
- Installation on macOS

### HCL Basics
- Blocks (resource, provider, data, variable, output)
- Arguments and meta-arguments
- Data types (string, number, bool, list, map, object)
- Expressions and interpolation
- Comments

### Providers
- Provider configuration
- AWS provider setup
- Authentication methods
- Multiple provider instances (aliases)

### First Configuration
- Writing resource blocks
- Data sources for dynamic values
- Resource dependencies
- Tags and naming conventions
