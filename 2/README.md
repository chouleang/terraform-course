# Day 2: Variables, Outputs, State Management & Modules

## Overview

This directory contains the Day 2 hands-on labs for learning advanced Terraform concepts.

**Duration:** 6-8 hours  
**Split into:**
- **2_1/** - Part 1: Variables, Outputs, Local Values (2-3 hours)
- **2_2/** - Part 2: Remote Backends, State Management (2-3 hours)
- **3/** - Day 3: Modules and Reusable Code (2-3 hours)

## What You'll Learn

### Part 2_1: Variables, Outputs & Locals
- ✅ Input variables with validation
- ✅ Variable types and defaults
- ✅ Output values (single, map, complex)
- ✅ Local values for calculations
- ✅ Sensitive data handling
- ✅ Variable files (.tfvars)
- ✅ Modifying and updating infrastructure

### Part 2_2: Remote Backends & State
- ✅ Remote state storage (S3)
- ✅ State locking (DynamoDB)
- ✅ State management commands (mv, rm, replace-provider)
- ✅ State file inspection
- ✅ Team collaboration patterns
- ✅ State file security best practices

### Day 3: Modules
- ✅ Module structure and organization
- ✅ Creating reusable modules
- ✅ Module inputs and outputs
- ✅ Terraform Registry
- ✅ Local vs registry modules
- ✅ Module best practices
